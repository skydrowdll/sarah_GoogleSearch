PLUGIN SARAH V3: GOOGLE SEARCH

le plugin GOOGLE SEARCH (pour Sarah v3) permet d'ouvrir une page internet avec un mot recherché.

Pour les fichier .ahk il faut télécharger le logiciel AutoHotkey en fonction de votre système d'exploitation

:::: INSTALATION ::::

(1) Vous devez avoir installé et configuré le plugin SCRIBE => https://github.com/tilleul/Sarah.Scribe

(2) Télécharger le plugin google search et installez le dans le dossier plugins.

(3) Changez la direction de votre navigateur par defaut dans le fichier ../bin/search.vbs EXAMPLE ( browser = """C:\Program Files (x86)\Google\Chrome\Application\chrome.exe""" )

:::: UTILISATION ::::

Dite "SARAH RECHERCHE VOITURE SUR INTERNET" suite a cela une page du navigateur s'ouvrira avec la recherche de voiture.
